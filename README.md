# Project Overview

AromaLux is a full-stack luxury perfume ecommerce platform built using the MERN stack. The platform provides a premium shopping experience for customers along with a professional admin dashboard for managing products, categories, users, and orders.

The application supports authentication, product management, image uploads, wishlist functionality, cart management, order tracking, forgot password flow, premium UI/UX, and production deployment.

# Environment Variables

## Backend (.env)

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
RAZORPAY_KEY_ID=your_razorpay_key
RAZORPAY_KEY_SECRET=your_razorpay_secret
CLOUDINARY_CLOUD_NAME=your_cloudinary_name
CLOUDINARY_API_KEY=your_cloudinary_key
CLOUDINARY_API_SECRET=your_cloudinary_secret
```
